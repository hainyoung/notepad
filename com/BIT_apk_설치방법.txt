1. new version apk 파일 로컬 폴더로 이동
-> 위치 복사

2. teamviewer old version에 속해있는 디바이스 열기(더블클릭)

3. 뜨는 창에서 파일 전송 클릭

4. apk 파일 위치 복사 붙여넣기 후 디바이스(폰)의 DCIM 폴더로 이동

5. 모두 전송되면 apk 파일을 열어 설치
(전송 완료 시 파랑색 아이콘 / 전송 중이면 초록색 아이콘)

(만약 설치에 실패하면 기존 어플 제거 후 new version apk 다시 열어서 재설치)

7. 설치 후 버전 정보 확인

8. 버전이 올바르면 teamviewer에서 새 버전 이름의 폴더로 디바이스를 이동

9. DCIM 폴더 내 apk 파일은 제거
