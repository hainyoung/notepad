# 1-3 지도학습의 빅픽쳐
지도학습을 하기 위해서는...
1) 과거의 데이터를 준비 
과거의 데이터 속에서 원인(독립변수)과 결과(종속변수)를 인식하는 것을 성공하면
나머지 복잡한 과정은 기계가 알아서 해 줄 것이다

2) 모델의 구조 만들기
온도를 알려주면 판매량을 예측하는 기계를 만들어보자
이런 기계를 머신러닝에서는 '모델'이라고 한다 

3) 데이터로 모델을 학습(FIT)
모델을 데이터에 딱 맞도록 만들어준다
이 과정을 FIT 이라고 한다
모델을 데이터에 맞게 FITTING 한다 == 학습

4) 모델을 이용

# 1-8 손실의 의미
모델은 매 학습마다 조금씩 학습을 함
각 학습이 끝날 때마다 그 시점에 모델이 얼마나 정답에 가까운지
평가하는 지표가 loss
학습이 진행될수록 점점 정답에 가까워질 것

loss가 0에 가까워질수록 학습이 잘 된 모델
epoch마다 loss가 0에 가까워지고 있는지 확인하는 것이 중요
0에 가까워질 때까지 학습을 반복하면 된다
