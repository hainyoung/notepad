# 1. 오리엔테이션

데이터과학을 위한 놀라운 도구인 오랜지 수업
오랜지는 코드 없이
드래그 앤 드롭으로
표의 분석, 시각화, 머신러닝을 이용한 예측과 같이 중요한 작업을 쉽게 배우고 사용할 수 있는

통계, 데이터마이닝, 데이터 과학이라 불리는 분야에서 사용할 수 있는 도구이다
오랜지는 표를 다루는 도구이다

오랜지를 이용해 표와 친해지는 여행을 시작
표, 시각화, 통계, 머신러닝 분야에서의 중요한 개념들을 이해하게 될 것

# 2. 시나리오 전략
레모네이드 카페 사장 이야기
이 이야기에서의 문제를 오랜지라는 도구를 이용해서 해결하는 과정을 겪을 것
이 도구의 본질은 무엇?
아무리 복잡한 정보 기술이라도 본질은 단순할 것
정보 기술의 본질은?
입력, 처리, 출력
어떤 분야에 입문하건 이것들을 제일 먼저 따져보자

오랜지라는 도구도 입력 처리 출력으로 이뤄져 있다
여기 해당하는 기능을 재빠르게 살펴보자
본질적 기능 + 부가적 기능도 살펴볼 예정

표는 중요한데 표만으로는 부족할 경우가 있다
어떤 정보들이 연관성을 가지고 있을 때 그 연관성을 알기 위해서는
그래프 같은 것으로 표를 시각화 할 필요가 있다
오랜지로 할 수 있다
시각화로 데이터를 파악했다면
그 데이터를 바탕으로 미래를 예측하고 현재에 결정을 내릴 수 있어야 한다
이 작업을 기계에게 시킬 순 없나 == 머신러닝, 기계학습
오랜지를 사용해서 머신러닝을 이용하는 방법도 알아볼 것이다
