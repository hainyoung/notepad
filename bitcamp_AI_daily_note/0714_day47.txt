데이터베이스부터 만들자
데이터베이스 우클릭 새 데이터 베이스
"bitdb" 란 이름으로 하나 생성

bitdb - 테이블 - 우클릭 새 테이블

열 이름 : 컬럼명
데이터형식 :  ncahr(10) 문자 10글자
int, float 등등이 있다

Test01 로 저장 후
개체 탐색기에서 테이블 새로 고침하면
만든 데이터베이스가 나타난다
현재, 테이블의 와꾸만 잡힌 상태!

Test01 우클릭 - 상위 200개행 편집


# Table 이름 앞에 dbo.은 항상 있다
# 일단 붙는다는 것만 알아둬라

# select문 sql 가장 기본

select * from test01;
 # * : 모든 것, test01을 검색하겠다 
# ;을 찍으면 문장 끝의 의미(안 찍어도 되긴 하다)
# F5 실행하면 테이블이 출력된다

select 이름 from test01;
# 이름 컬럼만 출력된다

select 이름, 연봉 from test01;
# 이름과 연봉 컬럽만 출력된다

bit2 - 속성
일반 - 기본데이터베이스  - bitdb
서버 역할 sysadmin check
사용자매핑 - bitdb check
사용자매핑  - 데이터베이스역할멤버자격 :  db_owner check

SQL server로 로그인
오류발생 - 18456

https://hannom.tistory.com/195 참고하여 해결

1. 윈도우로 일다 ㄴ로그인
2. 서버에서 우클릭 - 속성
3. 페이선택 - 보안 - SQL Server 및 Windows 인증 모드(S)에 체크
4. 확인 누르면 다시 시작하라는 창이 뜬다
5. 작업관리자(단축키 Ctrl + Shift + ESC)를 열어서 서비스 탭
- MSSQLSERVER를 찾아서 우클릭 - 다시시작 누르면 된다
6. 접속 확인
